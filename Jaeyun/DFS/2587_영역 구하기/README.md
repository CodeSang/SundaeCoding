# 영역 구하기

https://www.acmicpc.net/problem/2583


### 문제

크기가 MxN인 모눈종이에 눈금에 맞추어 K개의 직사각형을 그릴 떄,
K개의 직사각형의 내부를 제외한 나머지 부분이 몇 개의 분리된 영역으로 나누어지는지 구하고,
각 영역의 넓이를 오름차순으로 정렬하여 출력하시오.


### 입력 

첫째 줄에 M, N, K가 주어진다. 
둘째 줄부터 K개의 줄에는 한 줄에 하나씩 직사각형의 왼쪽 아래 꼭짓점의 x, y와 오른쪽 위 꼭짓점의 x, y가 주어진다.

-------------
### 풀이 

0. 입력받은 직사각형 위치를 통해 MxN에서 직사각형 내부들을 체크해둔다.
1. 직사각형 내부가 아닌 곳을 탐색하고, 아닌 곳이 존재한다면 영역의 개수를 증가시킨다.
2. 방문한 위치를 직사각형 내부라고 체크하고, 해당 영역의 넓이를 증가시킨다.
3. 그 주변(상, 하, 좌, 우)에도 직사각형 내부가 아닌지를 탐색한다. 
4. 내부가 아닌 곳이 존재하면 방문하고, 2로 이동
5. 더이상 주변에 직사각형 내부가 아닌 곳이 없을 때, 1로 이동한다.
6. 다음에 위치하는 직사각형를 탐색하고 위 과정을 반복한다.
